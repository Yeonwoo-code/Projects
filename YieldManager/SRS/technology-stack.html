# 기술 스택 (Technology Stack)

이 프로젝트는 MATLAB을 기반으로 하는 GUI 애플리케이션 프레임워크로, 다양한 기술 스택을 사용하여 구현되었습니다. 주요 기술 스택은 다음과 같습니다:

# 1. MATLAB

  • MATLAB은 이 프로젝트의 핵심 개발 환경으로, 수학적 계산과 GUI 개발을 위한 강력한 기능을 제공합니다.
  • MATLAB GUI는 사용자 인터페이스를 생성하고, MATLAB App Designer와 GUI 요소들을 통해 직관적인 GUI를 제공합니다.

# 2. Singleton 패턴

  • 싱글톤(Singleton) 패턴은 App 클래스의 인스턴스가 하나만 존재하도록 보장하기 위해 사용됩니다.
  • 이 패턴은 애플리케이션의 상태를 중앙 집중화하고, 코드 유지보수성과 확장성을 개선하는 데 도움을 줍니다.

# 3. 이벤트 기반 프로그래밍

  • 이벤트 등록 및 실행을 위한 메커니즘은 register()와 execute() 메서드를 통해 구현되었습니다.
  • Dynamic Event Handling을 통해 GUI와 이벤트를 유연하게 연결하고 관리합니다.

# 4. 객체 지향 프로그래밍 (OOP)

  • **객체 지향 설계(OOP)**를 통해 코드의 재사용성을 높이고, 애플리케이션의 유지보수성을 향상시킵니다.
  • 핵심 로직과 GUI를 분리하여, GUI 변경이 애플리케이션 로직에 영향을 미치지 않도록 합니다.

# 5. containers.Map

  • **containers.Map**은 MATLAB의 해시 맵 구조로, 이벤트 관리에 활용하여 이벤트 검색 성능을 최적화합니다.

# 6. MATLAB App Designer

  • MATLAB App Designer는 GUI 개발을 위한 고급 도구로, 이 프로젝트에서 사용자 인터페이스를 디자인하는 데 사용되었습니다.
  • figure와 같은 GUI 요소들을 관리하고, 동적인 이벤트를 처리합니다.

# 7. disp()

  • disp() 함수는 GUI 종료 시 사용자에게 종료 메시지를 출력하는 데 사용됩니다.
  • 이 기능은 사용자 경험을 향상시키고 애플리케이션 종료 시 명확한 안내를 제공합니다.
